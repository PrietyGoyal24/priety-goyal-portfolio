## 🎮 Quick Start Guide

```bash
# 📥 Clone the magic
git clone https://github.com/PrietyGoyal24/priety-goyal-portfolio.git

# 📂 Enter the project world
cd priety-goyal-portfolio

# 📦 Install the essentials
npm install

# 🔥 Fire up the development server
npm start

# 🏗️ Build for the universe
npm run build
```
